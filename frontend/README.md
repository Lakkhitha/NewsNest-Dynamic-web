# NewsNest Frontend

## Quick Start

1. Copy `.env.example` to `.env`.
2. Install dependencies:
	- `npm install`
3. Run app:
	- `npm run dev`
4. Build production:
	- `npm run build`

## Features Implemented

- Authentication UI (login/register)
- Dynamic timeline feed with refresh randomization
- News post composer
- Reactions and comments
- Admin dashboard (stats + report moderation)
- Mobile responsive layout
